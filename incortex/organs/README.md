# `organs/` — Organ Layer

An Organ is a specialized subsystem made of Tissues and Cells (Language, Memory, Reasoning, Planning, Learning, Speech, Safety, Tool, Development).

Planned modules (Design_Doc §12, §20):

- `base_organ.py` — shared `BaseOrgan` interface
- `input_organ.py`, `language_organ.py`, `memory_organ.py`, `reasoning_organ.py`,
  `planning_organ.py`, `learning_organ.py`, `speech_organ.py`, `safety_organ.py`, `tool_organ.py`

**Status:** scaffolding only — built in Phase 3 (Development Organ in Phase 8).
